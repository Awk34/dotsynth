Instructions

Create Nodes:
  • Hold blank screen space to bring up the new node menu
  • Tap a node on the menu to place it
Manipulate Nodes:
  • Drag nodes across the screen to move them
  • Tap nodes to bring up their parameters
    • Tap/Drag parameters to change them
Link Nodes:
  • Long-press and then drag from one node to another node to create a connection
    • Start at output, drag to input
